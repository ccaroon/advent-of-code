## Development
1. `pip install hatch`
2. `hatch shell`
3. ...write code...
4. `inv run <day> <part> [--test] [--debug] [--arg name=value]*`
    * Example: `inv run day01 1`
5. ...repeat...
6. `exit` to exit `hatch shell`
